# Sistema de Gerenciamento de Alunos

## 📌 Sobre o Projeto

Este é um sistema desktop desenvolvido em **C# (Windows Forms)** para o gerenciamento de alunos. A aplicação permite realizar operações completas de **CRUD (Create, Read, Update e Delete)**, oferecendo uma interface simples, organizada e intuitiva.

A navegação do sistema é baseada em componentes como **MenuStrip** e **ToolStrip**, facilitando o acesso às funcionalidades principais.

---

## 🚀 Funcionalidades

- Cadastro de usuário
- Verificação de Login
- Cadastro de alunos
- Consulta de registros
- Atualização de informações
- Exclusão de alunos
- Interface gráfica amigável
- Navegação estruturada com MenuStrip e ToolStrip

---

## 🖥️ Tecnologias Utilizadas

- C#
- Windows Forms (WinForms)
- .NET Framework / .NET (dependendo da versão do projeto)
- Arquitetura CRUD

---

## 📂 Estrutura do Projeto

O projeto é organizado de forma a facilitar manutenção e escalabilidade:

- **Forms**: Telas principais do sistema
- **Models**: Classes de representação dos dados (Aluno)
- **Services / Repository (se aplicável)**: Lógica de manipulação de dados
- **Resources**: Ícones e elementos visuais

---

## 🎯 Objetivo

O objetivo deste sistema é praticar e demonstrar conhecimentos em desenvolvimento desktop com C#, focando em:

- Manipulação de dados
- Interface gráfica com WinForms
- Organização de código
- Aplicação de padrões CRUD
